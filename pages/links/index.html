关于 "Friends of SliverRiver"

此页用途？​

在同时运营两个博客性质的站点的情况下，仅将友链放在单一站点已经不合理了，双站同步可能会信息错位。因此，我选择将友链页面移入 SliverRiver Library，并在双站设置友链页面跳转。

这看起来貌似不合理，实际是将两站的友链合一，让访客能够有机会访问各个与双站有友链交换的站点的机会。

是否开源？​

是的，它是开源的，并且内容与 SliverRiver's Hub 一致。您可以在 SliverRiver-Network/Library 看到对应的内容。

如何交换友链？​

优先条件​

或许大家都知道 友链友链，先友后链 这句话吧？虽然我很想交一些新朋友而不强求先友后链，但对于老朋友当然是有效的。

嗯，就这样。如果是属于银河眼熟的人都可以无视下面的大部分规则、顺利交换友链哒///(^v^)\\

基础要求​

  • 站点内容：
    • 文章多为原创，可少量转载；
    • 没有过于敏感，比如摄政（懂的都懂）、颜色、无脑地域黑等等；
  • 博客类型：独立博客，不是 CSDN、新浪博客这种类型的；
  • 域名：要求为独立域名（支持二级域名），亦或是带有身份特征的免费域名
    • 独立域名泛指 restent.win / restent.cf 这样的、由本人持有的域名；
    • 带有身份特征的免费域名泛指 restent.js.org 这样的、由本人完全控制的免费二级域名；
    • 不支持的域名类似 114514.coding-pages.com 或 restent.github.io/blog（即免费域名 + 二级目录）。

如何交换​

首先，将本站信息添加到你的友链列表里（排名先后无所谓）；

我的友链信息

标题：

  • 如果需要填写站点名，请填写你需要交换友链的站点（例如：Galaxy City）

  • 如果需要填写站长名，你可以任意去填，我的通用名字是 Restent Ou (Only Galaxy City) / SliverRiver / Neko3327 (Only Galaxy Sky)

Slogan：请根据站内顶部 Slogan 填写

链接：请根据你访问的站点域名填写

Favicon：https://library.restent.win/images/icons/favicon.png

头像：

  • Restent Ou: https://library.restent.win/images/icons/restent.jpg

  • Neko3327: https://library.restent.win/images/icons/neko3327.jpg

  • SliverRiver: 上方 Favicon

P.S. 请严格按照上述要求进行填写，身份信息不可混淆使用。

添加完成之后，你有两种办法添加：

发送电子邮件​

向 i@restent.win 发送一封电子邮件。标题任意，内容如下：

SliverRiver:
  link: https://www.restent.win
  avatar: https://library.restent.win/images/icons/favicon.png
  description: 追寻，光芒，救赎，命运

上述是一个示例。其对应的内容是：

  • SliverRiver -> 名称；

  • link -> 链接；

  • avatar -> 头像（链接）；

  • description -> 描述。

手动添加​

请点击这里 访问友链页面源代码。并且，您可以修改的地方是从 273 行（标注开始的地方）到标注结尾。

文件里已给出相关格式，请严格按照 HTML 代码格式标准以及上面相关信息提示进行添加。

何时通过？​

我一般有时间就会查看您的申请。申请合理后我会通过。:)
